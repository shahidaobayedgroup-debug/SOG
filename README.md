# Shahida Obayed Group - Corporate Website

This is the official corporate website for the Shahida Obayed Group (SOG). It is a static website built with HTML, CSS, and JavaScript, using the Bootstrap framework.

## Project Structure

The project follows a standard structure for a static website:

-   **index.html**: The main landing page.
-   **about.html**: Information about the company, its mission, vision, and history.
-   **sktml.html**: A page dedicated to the textile division of the group.
-   **contact.html**: Contact information and a form to get in touch.
-   **assets/**: Contains all the static assets for the website.
    -   `assets/brand/`: Logos and brand-related images.
    -   `assets/css/`: CSS files.
    -   `/assets/images/`: Images used throughout the site, organized by category.
    -   `assets/js/`: JavaScript files.
-   **custom-style.css**: Custom stylesheets for the website.
-   **style.css**: Main stylesheet for the website.

## Getting Started

To run this project locally, you can simply open the `index.html` file in your web browser. No special setup is required.

## Deployment

This is a static website, so it can be deployed to any static web hosting service, such as:

-   GitHub Pages
-   Netlify
-   Vercel
-   AWS S3

To deploy the site, simply upload the contents of this directory to your hosting provider.
